Introduction to EasySplitter_Merger

Easy Splitter is a Java based tool for making our distribution of files easier in removable media like external drive, ZIP drives etc., For this it can split a large file that cannot fit into a single External Drive or other media to one or more files that is easily or perfectly fit into them. Also it is possible to merge them back in destination machine.

How to Split files?

For splitting one or more files, do the following:

  • First, select those files you want to split by clicking the Add button. You can also edit this list by using Remove or Remove All buttons.
  • Naturally, all files are applied the split configurations in default (120 KB each). Suppose you want to specify separate settings for each file, you have to double click on the file in the list, Now Set button appears on the right. Now you can select any predefined size or customized size or number of splits and click Set button to save it.
  • Follow the same for other files also, if necessary.
  • Next choose the destination path where your split files needs to be created, either type the path directly or click Browse to select it Select the three options on the right, if necessary
  • Type any comments you want on the text area in the left, However it is applied only selecting one file to split. Now click the Split button to make the split.

How to Merge Files?

For merging the split files, do the following:

  • Select the Easy Splitter Info file (*.jfs) by clicking Browse button at the top.
  • Select the destination path where the spitted files need to be merged.
  • Click View Details to view the spitted file details.
  • Click Get Comments button to view the comments if any.
  • Click Merge button to do the Merge. It is important to have all the split files in the same folder as that of the *.jfs file.
  • Check Delete Split files after Merging if you want to do so before click the Merge button.
